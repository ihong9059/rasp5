# NVIDIA Jetson Orin Nano 소개

## 1. 개요

NVIDIA에서 만든 **소형 AI 컴퓨터**입니다. 라즈베리파이와 비슷한 크기지만 **CUDA GPU가 내장**되어 있습니다.

---

## 2. 라즈베리파이 5 vs Jetson Orin Nano

| 항목 | 라즈베리파이 5 | Jetson Orin Nano |
|------|---------------|------------------|
| **가격** | ~$80 | ~$200-500 |
| **CPU** | ARM Cortex-A76 (4코어) | ARM Cortex-A78AE (6코어) |
| **GPU** | VideoCore VII (AI 불가) | **NVIDIA Ampere (1024 CUDA)** |
| **AI 성능** | CPU만 사용 | **40 TOPS** |
| **메모리** | 8GB | 8GB (공유) |
| **PyTorch/CUDA** | 불가능 | **가능** |
| **크기** | 신용카드 크기 | 비슷함 |
| **전력 소비** | 5-15W | 7-15W |

---

## 3. EasyOCR 성능 비교 (예상)

| 장치 | 번호판 인식 시간 |
|------|-----------------|
| 라즈베리파이 5 (CPU) | 10-15초 |
| Jetson Orin Nano (GPU) | **1-2초** |
| 데스크탑 RTX 3060 | 0.3초 |

| 모델 | AI 성능 | 가격 | 용도 |

| Jetson Orin Nano (8GB) | 40 TOPS | ~$200 | 입문/취미 |
| Jetson Orin NX | 100 TOPS | ~$400 | 중급 프로젝트 |

---

## 5. 장점

- 라즈베리파이처럼 GPIO 핀 있음
- Linux (Ubuntu) 기반
- PyTorch, TensorFlow 네이티브 지원
- CUDA 가속 사용 가능
- 저전력 (7-15W)
- 소형 폼팩터

---

## 6. 단점

- 가격이 2-3배 비쌈
- 커뮤니티가 라즈베리파이보다 작음
- 설정이 조금 더 복잡함
- 일부 라즈베리파이 HAT와 호환 안 됨

---

## 7. 라즈베리파이 GPU 대안 비교

| 옵션 | 장점 | 단점 | EasyOCR 호환 |
|------|------|------|-------------|
| **Jetson Orin Nano** | CUDA 네이티브 | 가격 높음 | 완벽 호환 |
| Google Coral TPU | 저렴, USB 연결 | 모델 변환 필요 | 별도 작업 필요 |
| Hailo-8 | 고성능 | 모델 변환 필요 | 별도 작업 필요 |
| GPU PC (원격) | 최고 성능 | 네트워크 필요 | 완벽 호환 |

---

## 8. 현재 프로젝트에 적합한 경우

### Jetson 권장
- 실시간 번호판 인식이 필요할 때
- 여러 차량을 빠르게 처리해야 할 때
- 독립형(서버 없이) 시스템이 필요할 때
- 1-2초 이내 응답이 필수일 때

### 라즈베리파이 5 유지 권장
- 현재 10-15초 처리 시간이 괜찮을 때
- 예산이 제한적일 때
- 학습/프로토타입 목적일 때
